# 一、常规方法导入书源

提示

包括网络导入、二维码导入、本地导入

以网络导入为例，二维码导入，本地导入导入步骤相同

  1. 打开【我的】界面，选择【书源管理】选项

  2. 打开【更多】，选择【网络导入】

  3. 粘贴书源地址，点击【确定】

  4. 调整要导入的书源，点击【确定】按钮

# 二、常规方法导入替换净化源

提示

包括网络导入、二维码导入、本地导入

以网络导入为例，二维码导入，本地导入导入步骤相同

  1. 打开【我的】界面，选择【替换净化】选项
  2. 打开【更多】，选择【网络导入】
  3. 粘贴书源地址，点击【确定】
  4. 点击【确定】按钮

# 三、常规方法导入订阅源（RSS）

提示

包括网络导入、二维码导入、本地导入

以网络导入为例：

  1. 打开【订阅】界面，选择【更多】选项
  2. 打开【更多】，选择【网络导入】
  3. 粘贴书源地址，点击【确定】
  4. 调整要导入的订阅源，点击【确定】按钮

# 五、规则订阅方式导入源

提示

包括书源、订阅源、替换规则

优点：适用于长期更新且链接长期有效的源地址（例如github和gitee分享的源）

缺点：阅读2.0及阅读3.0前期版本不适用

  1. 打开【订阅】界面，选择【规则订阅】选项
  2. 在弹框中，选择要添加的【类型】，填写【名称】，【url】。最后点【确定】
  3. 点击【刚添加的订阅】。即可导入对应的源
  4. 调整要导入的源，点击【确定】按钮

# 六、导入 TTS 源

提示

包括本地导入、网络导入

本地导入方法同理，这里用网络导入做示范

  1. 打开【书籍界面】，调出【菜单页面】，打开【朗读】
  2. 打开【朗读设置】
  3. 点击【朗读引擎】
  4. 打开【更多】-选择导入方式，这里我选择网络导入
  5. 输入网址后，点击【确定】
  6. 选择【音源】后点击【确定】退出界面

# 三、参数介绍

下面讲解详细的参数，用于细节优化

# layout_flexGrow

layout_flexGrow实现的效果就是水平居中。

# layout_flexBasisPercent

这个属性默认设置了几个不同的值，在加入layout_flexGrow后，其参数所对应的效果，可以看下方表格

参数值 效果
0.05 一排十个分组，可以当占位符用
0.1 一排七个分组
0.15 一排五个分组
0.2 一排四个分组
0.25 一排三个分组
0.3-0.45 一排两个分组
0.5-1 一排一个分组

可能不同手机的宽度，显示的效果也不一样，具体可以用下方的源查看效果
